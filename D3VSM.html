How to Use This Value Stream Map Generator

Step 1: Start with Customer Requirement

A field titled "Customer Requirement" is presented.
Enter the goal or request that initiates the process.

Step 2: Add Work Centres/Process

Click the "Add Work Centre" button.
Enter:

  • Work Centre Name
  • (Auto-calculated) Total Cycle Time and Total VAT (updated based on tasks)
  • Optional comments

Step 3: Add Tasks to a Work Centre

Inside each work centre, click "Add Task".
Provide for each task:

  • Task Name
  • Cycle Time
  • Value Added Time
  • Owning Team (auto-suggests previously used team names)
  • Automation Level (Automated / Partially Automated / Manual)
  • Optional comments

Step 4: Reorder Work Centres or Tasks

Drag and drop to rearrange tasks or work centres as needed.

Step 5: Visualize the VSM

Once tasks and work centres are added, a VSM chart auto-generates. It shows:

  • Workflow sequence
  • Automation level indicators
  • Team ownership via color
  • Lead Time Ladders (Cycle Time vs. Value Added Time)

Step 6: Export / Save

Use the buttons to:

  • Download Chart as PNG
  • Export VSM as JSON (to save your work)
  • Import VSM JSON (to load a saved file)
  • Toggle JSON View (show/hide raw data)

TIP: Use Sample

Try the "Load Sample VSM" button to understand how a full map looks.
